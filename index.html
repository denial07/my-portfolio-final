About Thrifty

Thrifty is a sustainable thrift marketplace connecting buyers, sellers, and donors through a seamless e-commerce experience. It includes user roles, an analytics dashboard, and full payment flow integration.

Key Features

  • User Authentication & Role-Based Access
  • Real-Time Admin Dashboard
  • Stripe Payment Integration
  • Search, Filters, and Sorting
  • Email Notifications with Nodemailer
  • Donations made easy

Development Workflow

  • Planning: Defined core features and mapped user roles (buyers, sellers, donors, admin).
  • Design: Created wireframes and UI mockups focused on simplicity and sustainability appeal.
  • Development: Used a modular structure with React (frontend) and Express.js (backend API).
  • Testing: Manual testing of filters, cart, payments, and admin dashboard using test data.
  • Deployment: Set up hosting, integrated Stripe live keys, and configured PostgreSQL via Neon.

Challenges Faced

  • Role-Based Access: Implementing different user roles with protected routes required custom middleware logic.
  • Stripe Integration: Handling asynchronous payment flows and validating webhooks was complex but rewarding.
  • Filtering Logic: Optimizing multi-filter product queries in Pythons Shelve while maintaining UI responsiveness.
  • Responsiveness: Ensuring consistent UI across desktop and mobile required fine-tuning layout and spacing with Bootstrap.
  • Email Notifications: Setting up Nodemailer with secure credentials and spam prevention took extra effort.

Outcome

Thrifty successfully delivered a seamless and sustainable e-commerce experience by integrating a clean UI, efficient cart system, and eco-conscious branding. Users were able to browse and shop for items effortlessly, with features like smart filtering, real-time cart updates, and Python/ Shelve-backed product storage ensuring a smooth user journey.
